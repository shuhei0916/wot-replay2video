## テスト方針（メモ）

外部プロセス（WoT・OBS・YouTube API 通信）のラッパーはテスト対象外。
モックにすると「モックのテスト」になり費用対効果が低い。
テストを書くのは外部依存のない純粋ロジックのみ
（parse_replay のパース、edit_video のクリップ選択、upload_youtube のメタデータ構築等）。
